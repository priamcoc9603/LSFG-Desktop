## Overview
**Lossless Scaling** is a lightweight yet powerful Windows application that allows you to upscale any game or program with excellent image quality. It features lossless integer scaling, advanced AI frame generation (LSFG), and multiple upscaling algorithms for sharper visuals and significantly higher FPS.

> [!Important]
> This tool is intended for personal use with games and applications you legally own.

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `LosslessScaling.exe` as Administrator
3. Set your game to **Borderless Windowed** mode
4. Enable desired scaling mode and LSFG
5. Enjoy sharper visuals and higher FPS

---
## Troubleshooting
| Problem | Solution |
|----------------------------|-----------------------------------------------|
| Tool does not detect game | Use Borderless Windowed mode |
| Visual artifacts | Adjust Flow Scale or switch to Performance mode |
| Low performance | Lower multiplier or disable heavy sharpening |
| Antivirus warning | Add the folder to exclusions |
| Black screen / crash | Run as Administrator and update GPU drivers |
